# Программа анализа массива чисел

Эта программа выполняет следующие действия:

1. Создаёт массив чисел.
2. Вычисляет минимальное и максимальное значение в массиве.
3. Вычисляет сумму и среднее арифметическое элементов массива.
4. Выводит результаты на экран.

Программа написана на C++.

## Пример работы

Массив: `{3, 7, -2, 10, 0, 5}`  

Вывод:

min = -2
max = 10
avg = 3.83
